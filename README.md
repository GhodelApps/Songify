## General info
A Firebase based Android Music Player application built on MVVM achitecture pattern.

## Technologies
Project is created with:
* Kotlin
* Firebase
* XML

## Libraries Used
The various Libraries Used in this application are:
* Exoplayer2
* Firebase
* Dagger-Hilt
* Viewpager 2
* Material Design
* Livedata
* Navigation Component
* Coroutines
* Timber
* Glide
